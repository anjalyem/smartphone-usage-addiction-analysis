# 📱 Smartphone Usage and Addiction Analysis Dashboard

## 📌 Project Overview

Smartphone Usage and Addiction Analysis Dashboard is an Excel-based data analytics project developed to understand smartphone usage behavior and addiction patterns.

The project transforms raw user data into meaningful insights using data cleaning, transformation, and interactive dashboard visualization techniques.

## 🎯 Objectives

- Analyze smartphone usage patterns
- Identify addiction levels among users
- Study screen time behavior
- Understand app usage trends
- Generate actionable insights from user data

## 🛠 Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Data Visualization

## 🔄 Data Cleaning Process

Performed data preparation using:

- Removed duplicate records
- Handled missing values
- Corrected data formats
- Created calculated columns
- Prepared clean dataset for analysis

## 📊 Dashboard Features

- Total users analysis
- Average screen time tracking
- Addiction level analysis
- Age group comparison
- App usage insights
- Interactive filters and charts

## 📈 Key Insights

The dashboard helps identify:

- High smartphone usage groups
- User behavior patterns
- Addiction risk trends
- Usage differences between categories
